# Code of Conduct

이 커뮤니티는 서로 존중하는 협업을 지향합니다.

## Our Standards

- 사실 기반으로 소통합니다.
- 공격적/비하/차별적 표현을 금지합니다.
- 리뷰에서는 사람보다 코드와 근거를 다룹니다.
- 초보 기여자도 안전하게 질문할 수 있어야 합니다.

## Unacceptable Behavior

- 인신공격, 혐오 발언, 괴롭힘
- 반복적인 소모성 논쟁 유도
- 타인의 기여를 조롱하거나 폄하하는 행위

## Enforcement

위반 사례는 이슈 또는 메일로 신고할 수 있습니다.
관리자는 사실 확인 후 경고, 제한, 차단 조치를 할 수 있습니다.

## Scope

- GitHub 이슈/PR/디스커션
- 저장소와 관련된 공개 커뮤니케이션 채널
