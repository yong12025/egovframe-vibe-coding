# Security Policy

보안 이슈 제보를 환영합니다.

## Supported Scope

현재 저장소는 문서/프롬프트 중심 저장소입니다.
다만 아래 항목은 보안 관점에서 중요하게 다룹니다.

- 잘못된 보안 권장사항
- 인증/인가 취약 패턴을 유도하는 프롬프트
- 민감정보 노출을 유도하는 예시

## Reporting

보안 이슈는 공개 이슈 대신 비공개 채널로 제보해 주세요.

제보 시 포함 항목

1. 영향 받는 파일 경로
2. 재현 단계
3. 예상 영향
4. 권장 수정안

## Response Target

- 72시간 내 1차 응답
- 재현 가능 이슈 우선 수정
- 수정 후 변경 내역 공개
