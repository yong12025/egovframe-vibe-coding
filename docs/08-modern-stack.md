# Modern Stack

전자정부프레임워크 v5 라인에서 고려할 수 있는 현대적 스택 요소를 정리합니다.

---

## Candidate Stack

- Spring AI
- RAG
- Redis Stack
- Istio
- OpenTelemetry

---

## 적용 원칙

1. 먼저 핵심 업무 요구와 연결합니다.
2. 운영 복잡도 증가 여부를 평가합니다.
3. 단계적 도입 계획을 수립합니다.

---

## 실무 체크

- 관측성(로그/메트릭/트레이스) 준비 여부
- 장애 시 롤백 전략 존재 여부
- 인프라 운영 역량과의 적합성
