# Convention Similarity Audit Samples

`prompts/review/12-convention-similarity-audit.md` 출력 형식을 따르는 샘플 리포트 5종입니다.

## Samples

1. `01-legacy-v43-java.md`
2. `02-boot3-spring-service.md`
3. `03-ts-ui-module.md`
4. `04-sql-heavy-batch.md`
5. `05-greenfield-initial.md`

## Usage

1. `tools/convention-recommend.sh`로 추천 프로파일을 먼저 확인합니다.
2. 샘플과 동일한 구조로 감사 리포트를 작성합니다.
3. PR 본문에 `Recommendation Contract`를 포함합니다.
