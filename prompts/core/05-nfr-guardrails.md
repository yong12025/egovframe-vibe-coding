# Prompt: NFR Guardrails

비기능 요구사항(보안/성능/운영)을 구현 가드레일로 정의해줘.

입력
- 버전 라인: `<VERSION_LANE>`
- 시스템 유형: `<SYSTEM_TYPE>`
- 비기능 요구: `<NFR_REQUIREMENTS>`

작업 지시
1. 보안/성능/운영 항목으로 NFR을 분류해.
2. 각 항목을 코드/설정/운영 규칙으로 변환해.
3. 위반 시 탐지 가능한 검증 항목을 정의해.
4. 릴리스 전 최소 기준(Go/No-Go)을 제시해.

출력 형식
1. NFR 가드레일 표
2. 코드/설정 반영 지점
3. 검증 항목 및 명령
4. 릴리스 최소 기준
5. 리스크와 가정
