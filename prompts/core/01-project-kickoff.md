# Prompt: Project Kickoff

아래 요구사항으로 전자정부프레임워크 프로젝트 초기 설계를 만들어줘.

입력
- 프로젝트명: `<PROJECT_NAME>`
- 버전 라인: `<VERSION_LANE>` (v4.3 또는 v5.0 beta)
- 시작점: `<STARTER_TYPE>` (Template/Boot/MSA/Batch/Mobile/AI)
- 도메인: `<DOMAIN>`
- 핵심 기능: `<REQUIREMENTS>`

작업 지시
1. 버전 라인을 먼저 잠가.
2. 공통컴포넌트 재사용 가능한 기능을 먼저 매핑해.
3. 신규 구현이 필요한 기능만 남겨.
4. 초기 디렉터리 구조와 모듈 책임을 제안해.
5. 2주 단위 실행 계획을 만들어.

출력 형식
1. 버전 잠금 결과
2. 공통컴포넌트 재사용 매핑표
3. 신규 구현 백로그
4. 파일/모듈 구조
5. 2주 실행 계획
6. 리스크/가정
7. 검증 체크 항목
