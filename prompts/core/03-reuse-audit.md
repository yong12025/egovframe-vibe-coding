# Prompt: Common Component Reuse Audit

아래 기능 요구를 공통컴포넌트 재사용 관점으로 감사해줘.

입력
- 버전 라인: `<VERSION_LANE>`
- 기능 요구: `<REQUIREMENTS>`
- 현재 모듈: `<MODULE_LIST>`

작업 지시
1. 기능별로 공통컴포넌트 재사용 가능 여부를 분류해.
2. 재사용 가능하면 적용 위치와 확장 포인트를 제시해.
3. 재사용 불가하면 신규 구현 근거를 명시해.
4. 최종적으로 개발 공수를 절감하는 우선순위를 제안해.

출력 형식
1. 재사용 가능/불가 매트릭스
2. 적용/확장 가이드
3. 신규 구현 근거
4. 우선순위 실행안
5. 검증 방법(체크 항목/명령)
6. 리스크와 가정

추가 제약
- 판단 근거를 기능별로 1줄씩 명시해.
- 정보가 부족하면 필요한 추가 입력을 최대 3개 질문으로 제시해.
