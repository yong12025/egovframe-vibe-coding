# Prompt: Batch Job Generator

대용량 처리를 위한 배치 잡을 생성해줘.

입력
- 버전 라인: `<VERSION_LANE>`
- 잡 이름: `<JOB_NAME>`
- 입력/출력: `<IO_SPEC>`
- 스케줄 정책: `<SCHEDULE_POLICY>`

작업 지시
1. Job/Step 설계를 제안해.
2. Reader/Processor/Writer를 구현해.
3. 재시도/스킵/중복실행 방지 정책을 반영해.
4. 장애 시 재처리 전략을 포함해.
5. 운영 모니터링 지표를 제안해.

출력 형식
1. 배치 아키텍처
2. 파일 목록
3. 코드
4. 운영/장애 대응 가이드
5. 실행/검증 명령
6. 리스크와 가정

추가 제약
- 재시도/스킵 기준값을 숫자로 제안해.
- 실패 배치 재처리 절차를 단계별로 작성해.
