# Prompt: Search Module

검색 모듈(키워드/필터/정렬/페이징)을 생성해줘.

입력
- 버전 라인: `<VERSION_LANE>`
- 검색 대상: `<SEARCH_TARGETS>`
- 검색 요구: `<SEARCH_REQUIREMENTS>`

작업 지시
1. 검색 조건 DTO를 설계해.
2. 동적 쿼리(조건/정렬/페이징)를 구현해.
3. 성능을 위한 인덱스 제안을 포함해.
4. 검색 실패/빈 결과 처리 정책을 포함해.

출력 형식
1. 검색 설계 요약
2. 파일 목록
3. 코드/설정
4. 실행/검증 명령
5. 리스크와 가정
