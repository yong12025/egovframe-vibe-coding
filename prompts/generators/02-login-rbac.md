# Prompt: Login + RBAC

아래 요구사항으로 로그인/권한관리 기능을 생성해줘.

입력
- 버전 라인: `<VERSION_LANE>`
- 인증 방식: `<AUTH_MODE>` (세션/JWT)
- 권한 모델: `<ROLE_MODEL>`
- 로그인 정책: `<LOGIN_POLICY>`

작업 지시
1. 인증 흐름을 시퀀스로 설명해.
2. 사용자/권한 데이터 모델을 정의해.
3. 로그인, 로그아웃, 권한 검증 API(또는 컨트롤러)를 생성해.
4. 권한 누락/만료/잠금 상태 예외를 구현해.
5. 감사 로그 포인트를 추가해.
6. 보안 테스트 시나리오를 생성해.

출력 형식
1. 인증/인가 아키텍처
2. 파일 목록
3. 코드
4. 보안 설정
5. 테스트 시나리오
6. 실행/검증 명령
7. 리스크와 가정

추가 제약
- 파일 목록은 경로 단위로 제시해.
- 권한 실패(403)와 미인증(401) 응답 차이를 명시해.
