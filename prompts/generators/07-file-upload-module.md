# Prompt: File Upload Module

첨부파일 업로드/다운로드 모듈을 생성해줘.

입력
- 버전 라인: `<VERSION_LANE>`
- 모듈명: `<MODULE_NAME>`
- 파일 정책: `<FILE_POLICY>` (확장자/용량/보관경로)
- 보안 요구: `<SECURITY_REQUIREMENTS>`

작업 지시
1. 업로드/다운로드 API 또는 컨트롤러를 생성해.
2. 파일 메타데이터 저장 모델을 정의해.
3. 확장자/용량 검증 로직을 포함해.
4. 접근 권한 체크와 감사 로그를 추가해.
5. 실패/롤백 처리 정책을 포함해.

출력 형식
1. 아키텍처 요약
2. 파일 목록
3. 코드/설정
4. 실행/검증 명령
5. 리스크와 가정
