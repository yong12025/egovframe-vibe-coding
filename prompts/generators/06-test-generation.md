# Prompt: Test Generation Pack

기존 모듈에 대해 테스트 코드를 생성해줘.

입력
- 버전 라인: `<VERSION_LANE>`
- 대상 모듈: `<MODULE_NAME>`
- 기존 코드 경로: `<CODE_PATHS>`

작업 지시
1. 단위 테스트 대상과 통합 테스트 대상을 분류해.
2. 정상/실패/경계값 케이스를 작성해.
3. 인증/인가가 있는 경우 권한 케이스를 추가해.
4. 테스트 데이터 준비/정리 전략을 포함해.

출력 형식
1. 테스트 전략
2. 생성 파일 목록
3. 테스트 코드
4. 실행 명령
5. 잔여 리스크
