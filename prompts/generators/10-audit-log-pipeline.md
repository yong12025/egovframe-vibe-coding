# Prompt: Audit Log Pipeline

감사 로그 수집/조회 파이프라인을 생성해줘.

입력
- 버전 라인: `<VERSION_LANE>`
- 로그 이벤트 범위: `<AUDIT_EVENTS>`
- 보존 정책: `<RETENTION_POLICY>`

작업 지시
1. 감사 이벤트 모델을 설계해.
2. 서비스 레이어에 로그 기록 포인트를 삽입해.
3. 검색/필터 기반 조회 API를 생성해.
4. 민감정보 마스킹 정책을 적용해.

출력 형식
1. 이벤트 모델
2. 파일 목록
3. 코드/설정
4. 실행/검증 명령
5. 리스크와 가정
