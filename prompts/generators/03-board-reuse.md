# Prompt: Board With Common Component Reuse

게시판 기능을 공통컴포넌트 우선으로 구현해줘.

입력
- 버전 라인: `<VERSION_LANE>`
- 게시판 요구사항: `<BOARD_REQUIREMENTS>`
- 확장 항목: `<EXTENSIONS>`

작업 지시
1. 공통컴포넌트 재사용 가능한 기능을 우선 분류해.
2. 확장이 필요한 지점만 커스터마이징해.
3. 첨부파일/검색/페이징/권한 체크를 포함해.
4. 운영 로그와 오류 처리를 추가해.

출력 형식
1. 재사용 매핑표
2. 커스터마이징 파일 목록
3. 코드
4. 검증 절차
5. 실행/검증 명령
6. 리스크와 가정

추가 제약
- 공통컴포넌트 재사용 판단 근거를 기능별로 명시해.
- 신규 구현이 필요한 항목은 이유를 1줄로 적어.
