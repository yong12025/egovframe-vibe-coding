# Prompt: Admin Dashboard Scaffold

관리자 대시보드 초기 스캐폴드를 생성해줘.

입력
- 버전 라인: `<VERSION_LANE>`
- 대시보드 지표: `<DASHBOARD_METRICS>`
- 권한 정책: `<ADMIN_AUTH_POLICY>`

작업 지시
1. 대시보드 조회 API를 설계/생성해.
2. 지표 집계 로직을 서비스로 분리해.
3. 권한 검증과 접근 제어를 포함해.
4. 운영에 필요한 오류/로그 포인트를 추가해.

출력 형식
1. 화면/API 구조 요약
2. 파일 목록
3. 코드/설정
4. 실행/검증 명령
5. 리스크와 가정
