# Prompt: Common Code Management

공통코드 관리 모듈을 생성해줘.

입력
- 버전 라인: `<VERSION_LANE>`
- 코드 그룹 요구: `<CODE_GROUP_REQUIREMENTS>`
- 관리 정책: `<CODE_POLICY>`

작업 지시
1. 공통코드 엔티티/DTO/Mapper를 생성해.
2. 코드 조회/등록/수정/비활성화 API를 생성해.
3. 캐시 전략이 필요한 경우 반영해.
4. 운영자 권한 체크를 포함해.

출력 형식
1. 데이터 모델
2. 파일 목록
3. 코드/설정
4. 실행/검증 명령
5. 리스크와 가정
