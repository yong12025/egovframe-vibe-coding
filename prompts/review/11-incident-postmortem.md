# Prompt: Incident Postmortem

장애 사후 분석(Postmortem) 문서를 작성해줘.

입력
- 버전 라인: `<VERSION_LANE>`
- 장애 개요: `<INCIDENT_SUMMARY>`
- 타임라인: `<INCIDENT_TIMELINE>`
- 영향 범위: `<IMPACT_SCOPE>`

작업 지시
1. 사실 기반 타임라인을 정리해.
2. 근본 원인과 기여 요인을 분리해.
3. 재발 방지 액션을 단기/중기/장기로 제시해.
4. 액션별 담당/기한/검증 방법을 제안해.

출력 형식
1. 사건 요약
2. 타임라인
3. 근본 원인 분석
4. 재발 방지 액션
5. 검증 방법
6. 리스크와 가정
