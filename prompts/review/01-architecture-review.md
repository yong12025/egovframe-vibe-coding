# Prompt: Architecture Review

아래 코드/구조를 전자정부프레임워크 관점에서 리뷰해줘.

입력
- 버전 라인: `<VERSION_LANE>`
- 대상 경로: `<TARGET_PATHS>`
- 요구사항: `<REQUIREMENTS>`

작업 지시
1. 레이어 분리 위반 여부를 찾고 근거를 제시해.
2. 버전 혼합 가능성을 점검해.
3. 공통컴포넌트 재사용 누락을 점검해.
4. 개선 우선순위를 P1/P2/P3로 제안해.
5. 이슈마다 파일 경로와 가능한 라인 번호를 붙여.

출력 형식
1. 주요 이슈 목록
2. 영향도/우선순위
3. 수정 제안
4. 빠른 개선 순서
5. 즉시 조치 항목(오늘 처리 가능)
6. 리스크와 가정
