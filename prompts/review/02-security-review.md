# Prompt: Security Review

아래 모듈의 보안 취약점을 점검해줘.

입력
- 버전 라인: `<VERSION_LANE>`
- 대상 모듈: `<MODULE_NAME>`
- 코드 경로: `<CODE_PATHS>`

작업 지시
1. 인증/인가 누락 지점을 식별해.
2. 입력 검증, SQL Injection, XSS, CSRF 위험을 점검해.
3. 로그 민감정보 노출 여부를 점검해.
4. 즉시 조치 가능한 수정안을 코드 단위로 제안해.
5. 이슈마다 파일 경로와 가능한 라인 번호를 붙여.

출력 형식
1. 취약점 목록
2. 위험도
3. 수정안
4. 재검증 체크리스트
5. 즉시 조치 항목(오늘 처리 가능)
6. 리스크와 가정
