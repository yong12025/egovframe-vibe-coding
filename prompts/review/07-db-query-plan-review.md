# Prompt: DB Query Plan Review

쿼리 실행 계획 기반으로 DB 성능을 리뷰해줘.

입력
- 버전 라인: `<VERSION_LANE>`
- 대상 쿼리: `<SQL_QUERIES>`
- 실행 계획: `<QUERY_PLANS>`

작업 지시
1. Full Scan/불필요 조인/정렬 비용 과다 여부를 식별해.
2. 인덱스/쿼리 리라이트 개선안을 제시해.
3. 데이터 분포 변화에 따른 리스크를 분석해.
4. 개선 후 재측정 절차를 제시해.

출력 형식
1. 문제 쿼리 목록
2. 개선안
3. 재측정 방법
4. 리스크와 가정
