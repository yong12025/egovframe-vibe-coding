# Prompt: Observability Review

로그/메트릭/트레이스 관점으로 관측성을 리뷰해줘.

입력
- 버전 라인: `<VERSION_LANE>`
- 대상 모듈: `<MODULE_NAME>`
- 관측 설정: `<OBSERVABILITY_CONFIG>`

작업 지시
1. 핵심 유스케이스 추적 가능 여부를 점검해.
2. 로그 컨텍스트/에러 상관관계 부족 지점을 식별해.
3. 메트릭/알람 누락 항목을 제시해.
4. 운영 장애 대응 시간을 줄이는 개선안을 제시해.

출력 형식
1. 관측성 점검 결과
2. 누락 항목
3. 개선안
4. 검증 방법
5. 리스크와 가정
